# KAIROS Design System

## Brand

KAIROS is a light, readable intelligence dashboard for the moment before consensus.

Core line:

> The moment before consensus.

Japanese line:

> 未来が後から証明する、現在の分岐点を記録する。

Brand explanation:

> Kairos is not ordinary time. It is the decisive instant hidden inside ordinary days. KAIROS watches facts, capital, and narratives until a pattern becomes visible before consensus.

## Visual Direction

- Light background, high readability.
- Minimal, calm, editorial.
- The feeling should be closer to a research memo and intelligence cockpit than a dark trading terminal.
- Use color only to signal meaning.

## Palette

- Background: `#FAF9F5`
- Surface: `#FFFFFF`
- Secondary surface: `#F2EFE8`
- Text: `#151515`
- Muted text: `#667085`
- Line: `#E4DFD4`
- Gold accent: `#B8872C`
- Blue signal: `#2563EB`
- Green confirmation: `#0F8A5F`
- Red risk: `#C2410C`

## Product Language

Use:

- Kairos Moment
- Ordinary time
- Decisive instant
- Future Ledger
- Future Receipt
- At The Time
- Signal
- Mechanism
- Capital Path
- Market Consensus
- Non-consensus View
- Reality Check
- Falsifier

Avoid:

- Guaranteed returns
- Stock picks
- Explosive profit
- Any wording that implies investment advice

## UI Priorities

1. Readability.
2. Clear analysis structure.
3. Evidence and interpretation separation.
4. Future review built into every card.
5. Simple static files that can later become Next.js data routes.

## Evidence UI

KAIROS must always show the raw facts behind the view:

- Source URL
- Source type
- Date
- Extracted fact
- Numeric fields for trades, filings, earnings, and market data

Never let an Oracle View appear without Evidence.
